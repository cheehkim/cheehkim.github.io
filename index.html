🌐 생성형 AI 및 도구 소개

Ollama
  • 주요 기능/간단한 설명: 로컬에서 다양한 LLM 모델을 설치·실행·관리할 수 있는 플랫폼
  • 사용법 & 다운로드 & 주의사항:
    - 운영체제용 설치 파일 다운로드 후 설치
    - 터미널에서 `ollama run <모델명> "<프롬프트>"` 명령으로 모델 자동 다운로드 및 실행
    - 대형 모델은 GPU 메모리 요구량 높음, CPU 전용 시 성능 저하 주의
  • 제공사 & 유/무료 여부: Ollama Inc., 플랫폼 및 기본 모델 무료, 상업용 고급 모델은 별도 과금
CustomGPT
  • 주요 기능/간단한 설명: 맞춤형 GPT 모델을 생성·배포하는 도구
  • 사용법 & 다운로드 & 주의사항:
    - GitHub 레포 클론 또는 웹사이트 접속 → 데이터셋 업로드 → 파인튜닝 설정 → API 키 입력
    - 무료 체험 후 구독 전환 가능
    - API 키 관리 주의, 개인정보 보호
  • 제공사 & 유/무료 여부: OpenAI·Hugging Face, 무료 사용량·종량 과금
